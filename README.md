##  Project Overview

This project was done as part of the Digital Literacy course. In this assignment, I took the role of a Student Digital Ambassador and worked on different tasks to understand how to use digital tools properly and safely.

The main aim of this project is to learn about online platforms, digital communication, and cyber safety. All the tasks are completed and organized in this repository along with the project report.

## Task Summary

###  Task 1: Digital Literacy Infographic
In this task, I created an infographic using Canva. It explains the meaning of digital literacy, safe internet practices, and useful tools for students. This task helped me understand how to present information in a simple visual way.

---

###  Task 2: Digital Portfolio
In this task, I created and updated my profiles on platforms like GitHub and LinkedIn. I learned how important it is to have a professional online presence for future opportunities.

---
